## Getting Started

**From GitHub Releases**

```bash
$ wget https://github.com/hivelocityinc/adminize-boilerplate/releases/download/v0.3.0/boilerplate.zip
$ unzip boilerplate.zip
$ cd boilerplate
$ bower install
```
